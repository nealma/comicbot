## 嗨，欢迎！👋

你好！我们是**码上同行**的作者——一个程序员爸爸和他好奇的女儿。

这个博客的名字"码上同行"有双重含义：
- **码** = 代码，代表我们一起写代码
- **马上** = 马上出发，代表我们迫不及待地开始这段旅程

## 为什么要写这个博客？

爸爸一直想找一个方式和女儿分享编程的乐趣。女儿呢，也对"爸爸每天在电脑前敲的那些东西"充满了好奇。

于是，我们决定：**一起学，一起写，一起成长！**

## 我们要学什么？

我们的计划包括：

1. **网页开发** - HTML、CSS、JavaScript
2. **React** - 构建交互式界面
3. **Python** - 数据科学和小游戏
4. **算法** - 用有趣的方式理解计算思维

## 第一段代码

让我们从最经典的程序开始：

```javascript title="hello.js"
// 我们的第一个程序！
function greet(name) {
  return `你好，${name}！欢迎来到码上同行 🚀`;
}

console.log(greet("世界"));
console.log(greet("每一位读者"));
```

```python title="hello.py"
# Python 版本
def greet(name):
    return f"你好，{name}！欢迎来到码上同行 🚀"

print(greet("世界"))
print(greet("每一位读者"))
```

> 💡 **小贴士**: 编程最重要的不是语法，而是**解决问题的思维方式**。每一种编程语言都只是工具，思维才是核心！
